# 📘 Методи підвищення швидкодії нейронних мереж за допомогою карт динамічних режимів

Проєкт присвячений дослідженню впливу гіперпараметрів адаптивних оптимізаторів на швидкодію та стабільність навчання нейронних мереж за допомогою карт динамічних режимів.

## 🧠 Опис функціоналу

Основні можливості програми:

- Завантаження та підготовка наборів даних (CIFAR-10, CIFAR-100, Fashion-MNIST)
- Перетворення кольорових зображень у відтінки сірого
- Побудова MLP-мережі без використання високорівневих ML-фреймворків
- Реалізація оптимізаторів Adam, AdamW та AdaMax
- Побудова карт динамічних режимів у просторі гіперпараметрів β₁ та β₂
- Аналіз зон збіжності, коливань і розбіжності
- Збереження результатів експериментів та візуалізацій

Результатом роботи програми є карти динамічних режимів, які дозволяють візуально визначати області параметрів, що забезпечують найбільш швидке та стабільне навчання нейронної мережі.

## 🧪 Експериментальні параметри

### 📊 Набори даних
- CIFAR-10
- CIFAR-100
- Fashion-MNIST

### 🧠 Архітектура моделі
- MLP (для CIFAR-10 та CIFAR-100)
- Simple CNN (для Fashion-MNIST)
- Hidden size: 20, 28, 30, 40, 50
- Activation: sigmoid
- Batch size: 64

### ⚙️ Гіперпараметри оптимізаторів
- Learning rate (α): 0.01, 0.1, 0.2, 0.3, 0.4, 0.6
- β₁: 0.0–1.0 (step 0.005)
- β₂: 0.0–1.0 (step 0.005)

### 🔁 Умови експерименту
- Epochs per point: 5, 10, 15, 20, 25, 30, 40

## 🧪 Проблеми і рішення

| Проблема                              | Рішення                                      |
|---------------------------------------|----------------------------------------------|
| Нестабільність при високому α         | Локалізація стабільних параметрів β₁ та β₂ за допомогою карт режимів |
| Розбіжність навчання                  | Зменшення learning rate                      |
| Великі витрати часу на генерацію карт | Зменшення кількості епох або розміру вибірки |
